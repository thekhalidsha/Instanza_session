# Instanza_session
A Python Script for Generating Instagram Login Session Cookies for use in Instanza

### How to?
Clone or Download the Git Repo

Packages Required
```
pkg install python3
```
Requirements to be Satisfied
```
pip install -r requirements.txt
```
Run Instanza Session
```
python3 app.py
```
